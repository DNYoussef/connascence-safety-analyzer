### Information Security Program

**Q: Does your organization have a formal information security program?**  
**A:** Yes. Connascence Systems maintains a comprehensive information security program based on ISO 27001:2022 standards with annual third-party audits.

**Q: Do you conduct regular security awareness training?**  
**A:** Yes. All employees complete quarterly security awareness training with annual phishing simulation testing. Completion rate: 100%.

**Q: How do you handle security incidents?**  
**A:** 24/7 Security Operations Center (SOC) with defined incident response procedures. Mean time to detection: < 15 minutes. Mean time to response: < 1 hour for critical incidents.

### Access Controls and Identity Management

**Q: What access control mechanisms do you use?**  
**A:** Multi-factor authentication (MFA) required for all system access. Role-based access control (RBAC) with principle of least privilege. Regular access reviews conducted quarterly.

**Q: How do you manage privileged access?**  
**A:** Privileged Access Management (PAM) solution deployed. All privileged activities logged and monitored. Just-in-time access for administrative functions.

### Data Protection and Encryption

**Q: What encryption standards do you use?**  
**A:** AES-256 for data at rest, TLS 1.3 for data in transit. Hardware Security Modules (HSMs) for key management. Annual cryptographic review by external security firm.

**Q: Where is customer data stored?**  
**A:** Customer choice of AWS, Azure, or GCP regions. Data never crosses jurisdictional boundaries without explicit consent. Local data processing options available.

### Compliance and Certifications

**Q: What compliance certifications do you maintain?**  
**A:** SOC 2 Type II (annual), ISO 27001:2022, ISO 9001:2015, GDPR compliance. FedRAMP Moderate authorization in process.

**Q: Do you undergo regular third-party security assessments?**
